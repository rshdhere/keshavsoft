Bootstrap learnings
  • Grid first: use container, row, col-* for layout.
  • Utilities win: spacing (p-*, m-*), display, flex, text.
  • Components are composable: cards, navbars, modals, forms.
  • Responsive by suffix: -sm, -md, -lg, -xl.
  • JS via data attributes: data-bs-toggle, data-bs-target.
